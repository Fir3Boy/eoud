# eoud
eOud – Interactive Fragrance Experience
An interactive, multi-language demonstration platform for eOud, a premium Saudi Arabian agarwood (oud) product by Surrati Perfumes. This project combines traditional heritage with modern web technology to showcase the art of incense.

🌟 Features
14-Language Support: Localized experiences for the Middle East, South Asia, Africa, and East Asia, including Arabic, Farsi, Urdu, Hindi, Bengali, and more.

Interactive Smoke Simulation: A custom-built CSS and SVG animation engine that simulates realistic rising incense smoke.

GSAP Powered Animations: High-end scroll-triggered animations using GSAP and ScrollTrigger to reveal fragrance ingredients as the user explores the collection.

Regional Categorization: A specialized landing page that organizes languages by geographical region for easier navigation.

🛠️ Technology Stack
Frontend: HTML5, CSS3 (Custom animations and SVG filters).

Animation Libraries: GSAP (GreenSock Animation Platform) and ScrollTrigger.

Hosting: GitHub Pages.

🇸🇦 About the Product
eOud is a "Smart Incense" solution that requires no charcoal or traditional burners. Proudly made in Saudi Arabia, it draws on a legacy spanning nearly a century to provide a long-lasting, concentrated agarwood experience.

📂 Project Structure
/index.html — The main regional language selection (Splash Page).

/ar, /en, /ur, etc. — Individual localized demonstration pages.

/assets/ — Shared CSS, JavaScript (GSAP logic), and high-quality fragrance imagery.
